# 臨床実践・治療

この index は、臨床実践・治療サイトの公開入口です。治療法そのものだけでなく、適応判断、副作用・有害事象、支援体制、医療安全、危機対応までを横断して読むための導線を置いています。

## 主要MOC

- [[MOC｜臨床実践・治療]] - サイト全体の横断 MOC
- [[MOC｜心理療法]] - CBT、ACT、DBT、トラウマ焦点化治療、治療同盟、終結、適応判断
- [[MOC｜薬物療法]] - 向精神薬、抗うつ薬、抗精神病薬、気分安定薬、睡眠薬、TDM、副作用、減薬
- [[MOC｜神経調節・身体療法]] - ECT、rTMS、DBS、tDCS/tACS、運動、睡眠、栄養、身体技法
- [[MOC｜リハビリ・生活支援]] - ケースマネジメント、地域支援、就労・復職、生活機能、訪問看護
- [[MOC｜医療安全・危機対応]] - 自殺・自傷、興奮、暴力、身体急変、行動制限、インシデント分析
